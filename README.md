# FlaskLibrary

FlaskLibrary — це вебпроєкт на Flask для **публічної онлайн-бібліотеки**.

## Ідея проєкту
- надати відкритий каталог книжок для всіх користувачів;
- дати можливість переглядати книжки, профілі та відгуки;
- поступово розвивати сервіс у повноцінну публічну бібліотеку онлайн.

## Технології
- Python
- Flask
- SQLAlchemy
- Jinja2

## Швидкий старт
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Після запуску відкрийте застосунок у браузері за локальною адресою, яку покаже Flask.
